# 📝 React Native Todo App

A simple, clean todo app

## ✨ Features

- Add new todos
- Mark todos as done/undo
- Edit todo titles inline
- Delete todos
- Persistent storage with AsyncStoragex

## 🛠 Setup

```bash
git clone https://github.com/ayomidedaniel1/qoneqt.git

cd qoneqt

npm install

Then start the app:

npm run start
```
